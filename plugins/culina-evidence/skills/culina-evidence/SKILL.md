# Culina evidence

1. Find the authoritative requirement, protocol or component document.
2. Trace the claim into source and verification with exact references.
3. Classify evidence as static, host, simulator/model or physical product.
4. Mark assumptions and missing evidence instead of filling them with guesses.
5. Do not edit files or use repository history as an answer source.
