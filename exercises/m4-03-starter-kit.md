# Team transfer

Duration: 20 minutes

## Outcome

Each group leaves with a small adoption proposal based on evidence from the
day, not a bundle of speculative agent configuration.

## Task

1. Select at most three artifacts that materially helped today: a scoped rule,
   context package, verification workflow, reviewer definition, hook or gate.
2. For each artifact, record the problem it solved, repository-specific
   assumptions, owner and first real use case.
3. Remove Culina details before proposing it for a team repository.
4. Define one two-week experiment with an observable outcome.
5. Prepare a three-minute report of the feature evidence and adoption proposal.

## Done when

- Every proposed artifact has an observed reason to exist.
- The proposal names what must still be adapted or approved.
- The first experiment is small enough to start in the next normal work cycle.
